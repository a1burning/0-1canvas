setTransform：红色和蓝色都是旋转了相同的角度
transform：红色和蓝色都是基于前一个变换旋转的

setTransform方法

图形变换方法，和transform的方法类似

注意：transform方法的变换都是参考上一次的，而setTransform会重置图形状态后再改变的。

cxt.setTransform(a, b, c, d, e, f)

  • a: 水平缩放绘图
  • b: 水平倾斜绘图
  • c: 垂直倾斜绘图
  • d: 垂直缩放绘图
  • e: 水平移动绘图
  • f: 垂直移动绘图

setTransform-平移操作

cxt.setTransform(1, 0, 0, 1, e, f)

setTransform-缩放操作

cxt.setTransform(a, 0, 0, d, 0, 0)

setTransform-旋转操作

cxt.setTransform(cosθ, sinθ, -sinθ, cosθ, 0, 0)
