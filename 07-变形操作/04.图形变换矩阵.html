transform方法

变换矩阵方法，集合了平移、缩放和旋转，需要用到变换矩阵计算。

cxt.transform(a, b, c, d, e, f)

  • a: 水平缩放绘图
  • b: 水平倾斜绘图
  • c: 垂直倾斜绘图
  • d: 垂直缩放绘图
  • e: 水平移动绘图
  • f: 垂直移动绘图

transform-平移操作

cxt.transform(1, 0, 0, 1, e, f)

transform-缩放操作

cxt.transform(a, 0, 0, d, 0, 0)

transform-旋转操作

cxt.transform(cosθ, sinθ, -sinθ, cosθ, 0, 0)
