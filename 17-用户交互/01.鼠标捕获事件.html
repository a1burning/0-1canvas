捕获物体

canvas中对图形的捕获，和DOM元素的捕获是不一样的。

  • 矩形捕获: 鼠标点击的点落在矩形上
  • 圆形捕获: 鼠标点击的点距离圆形小于圆的半径
  • 多边形捕获和不规则图形捕获: 分离轴定理(SAT)和最小平移向量(MTV)

矩形的捕获语法

if (mouse.x > rect.x &&
    mouse.x < rect.x + rect.width &&
    mouse.y > rect.y &&
    mouse.y < rect.y + rect.height) {
    ...
}

圆形的捕获语法

dx = mouse.x - circle.x;
dy = mouse.y - circle.y;
distance = Math.sqrt(dx * dx + dy * dy);
if (distance < circle.radius) {
    ...
}
