三角函数

  • 正弦函数：sin(θ)
  • 余弦函数：cos(θ)
  • 正切函数：tan(θ)

语法

  • sin(θ)：Math.sin(θ * Math.PI / 180)
  • cos(θ)：Math.cos(θ * Math.PI / 180)
  • tan(θ)：Math.tan(θ * Math.PI / 180)
  • arcsin(x / R)：Math.asin(x / R) * (180 / Math.PI)
  • arccos(x / R)：Math.acos(x / R) * (180 / Math.PI)
  • arctan(x / R)：Math.atan(x / R) * (180 / Math.PI)

弧度制写法：度数 * Math.PI / 180

反正切函数的问题

我们可以使用反正切函数Math.atan()来求出两条边之间夹角的度数。

但是使用Math.atan()会出现一个度数回应两个不同象限角的问题。不是唯一确定的角。

所以这个时候需要用到另一个反正切函数Math.atan2()

反正切函数语法

Math.atan2(y, x)

  • y：对边的边长，且有正负之分
  • x：邻边的边长，且有正负之分

注意：我们可以使用反正切函数Math.atan2()求出两条边之间夹角的度数，还能够准确判断该度数对应的是哪一个夹角。

注意：这个函数可以实现追随鼠标指针旋转。
